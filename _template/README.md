# Project Template

Reference architecture for new echosystem projects.

## What this is

Not a runnable project. A decision guide and structural reference. When a new project is requested, open `CLAUDE.md` in this folder — it tells you exactly what to build and what to skip.

## How to use it

1. Read `CLAUDE.md`
2. Follow the decision guide — build only what the project needs
3. Add the project to the echosystem registry
4. Start building
